Blender

Урок 6 – 2D-мяч

В результате этого урока должна получиться следующая анимация:

Немного теории:

Blender известен тем, что отлично подходит для 3D, но знаете ли вы, что он также великолепен для 2D-анимации? Это относительно новая функция, представленная в Blender 2.8, и она включает в себя ряд уникальных функций и инструментов, которые очень хорошо работают для создания 2D-анимации. Создайте новый файл 2D-анимации, выбрав его во вступительном окне. Вы также можете перейти в Файл > Создать > 2D-анимация.

Это будет основной режим, который вы будете использовать при создании рисунков и анимации.

Большую часть рисунка мы создадим в режиме рисования Draw Mode. Это единственный режим, позволяющий создавать 2D-обводки. Уже созданные штрихи нельзя выбрать в режиме рисования. Если вы хотите редактировать обводки, вместо этого вы должны использовать режим редактирования Edit Mode или режим скульптуры Sculpt Mode.

Grease Pencil — это основной инструмент, который вы используется в Blender для создания 2D-анимации, и позволяет рисовать в 3D-пространстве.

Blender настоятельно рекомендует использовать планшет или сенсорный дисплей при использовании этого инструмента.

Чтобы создать новый объект Grease Pencil, сначала необходимо перейти в режим Object Mode, выбрав его в раскрывающемся меню в левом верхнем углу.

Затем перейдите в Add - Grease Pencil, чтобы увидеть несколько различных вариантов на выбор.
Blank (пусто): создает нулевой объект «Grease Pencil».
Stroke (обводка): создание примера стилизованной демонстрации обводки.
Monkey: Создает пример лица обезьяны.

Создайте пустой объект Grease Pencil, а затем вернитесь в рабочую область 2D-анимации, чтобы вернуться в режим рисования.

Слева вы найдете все основные инструменты, в том числе инструмент «Рисование», инструмент «Заливка», инструмент «Ластик» и т. д.

Базовый инструмент рисования будет включать все основные настройки, такие как мазок кисти и цвет, которые вы можете настроить в правой части экрана.

Выбрав Draw Tool, вы можете изменить кисть, щелкнув значок карандаша, расположенный в верхней строке меню.

В верхней панели у вас также есть другие возможности изменить основные настройки инструмента «Рисование», такие как цвет, радиус, сила и т. д.
Эти настройки также доступны в правой части экрана в разделе Active Tool & Workspace Settings.

Чтобы использовать инструмент Draw Tool просто щелкните и перетащите его, как если бы вы использовали любое другое программное обеспечение для редактирования фотографий. Штрихи жирным карандашом в Blender являются 2D, но они существуют как 3D-объект. Закончив рисование, вы можете редактировать эти штрихи дальше, перейдя в режим редактирования Edit Mode.

В Blender объект Grease Pencil можно анимировать следующими способами:
• Inherited Animation (унаследованная анимация): использование движения другого объекта в качестве родительского.
• Moving as a Whole Object (перемещение всего объекта): изменение положения, ориентации или размера
• Deforming (деформация): анимация их точек
• Drawing Frame by Frame (рисование покадрово): рисование по одному кадру за раз, как в традиционной анимации.

Создание анимации:

Чтобы запустить новую анимацию, выберите File > New > 2D Animation чтобы создать новый шаблон анимации. В этом шаблоне уже должен быть создан объект Grease Pencil и включены автоматические ключевые кадры. Нарисуйте шар и назовите его «Ball». Если хотите, вы также можете использовать для этого инструмент Circle Tool. Чтобы получить идеальный круг, вы можете нажать клавишу Shift, щелкая и перетаскивая.

Как только вы будете довольны формой, нажмите Enter на клавиатуре. Это создаст объект Grease Pencil и добавит ключевой кадр на временную шкалу. Вы также можете удалить заливку Fills, поскольку в этом уроке мы будем использовать только линии.

Создайте новый объект Grease Pencil для пола, перейдя в Object Mode, а затем выберите Add > Grease Pencil > Blank. Переименуйте его в «Floor», а затем нарисуйте пол в режиме рисования.

Используйте тот же процесс, чтобы нарисовать траекторию движения прыгающего мяча. Вы можете переименовать этот объект Grease Pencil в «PathGuide». Мы будем использовать это в качестве руководства для анимации вашего мяча.

Перейдите в Edit Mode и переместите мяч в начало вашего пути.

Выделите ключевой кадр для мяча и продублируйте его, нажав Shift+D. Затем переместите ключевой кадр вперед примерно на 10 кадров.

Сделайте это 6 раз, чтобы мяч отскочил 3 раза. Измените конечный кадр анимации, чтобы он заканчивался одновременно с последним ключевым кадром.

Перейдите в Edit Mode и переместите временную шкалу вперед к следующему ключевому кадру. Затем переместите мяч туда, где вы хотите, чтобы он находился в данный момент.

Повторяйте шаги для каждого ключевого кадра, пока не дойдете до конца анимации.

Теперь, если вы нажмете кнопку предварительного просмотра, вы увидите, что теперь у нас есть базовая анимация!
Чтобы сделать анимацию более плавной, вам нужно заполнить пространство между каждым из текущих ключевых кадров новыми ключевыми кадрами. Поэтому повторите тот же процесс, что и раньше, чтобы заполнить недостающие ключевые кадры.

Затем вы можете организовать свои ключевые кадры. Выберите ключевые кадры-заполнители, нажмите R на клавиатуре и выберите Breakdown, чтобы теперь вы могли увидеть визуальную разницу для разных типов ключевых кадров.

Добавьте некоторые окончательные настройки для каждого кадра, например, «сжатие» и «растягивание» формы шара, чтобы улучшить внешний вид анимации.

Отрендерим полученный результат.
