Урок 3 – NTF

В результате этого урока должна получиться следующая анимация:

Удалим все со сцены и добавим Iso Sphere.

В режиме редактирования выберем несколько вершин затем при помощи Amount of Adjacent Facesa (Shift+G) выделим грани.

Уменьшим масштаб, чтобы получилось что-то похожее на звезду.

К получившейся фигуре применим Shade Smooth.

Добавим модификатор Bevel. Выберем Limit Method = None, Amount = 0,008.

Добавим модификатор Simple Deform. Выберем Angle=360.

Добавим модификатор Subdivision Surface.

Перейдем во вкладку настройки шейдеров и добавим новый материал, который будет состоять из шейдеров Emission, Texture Coordinate.
Соединим Surface «Material Output» с Emission «Emission».
Normal «Texture Coordinate» соединим с Color «Emission».
В Emission установим значение Strenght=1.

Наведем камеру таким образом, чтобы в кадр попали цвета которые нам больше всего нравятся и немного увеличим масштаб.

Теперь, чтобы анимировать кадр, нужно разбить векторы на 3 части и проделать некоторые математические действия.

Для этого добавим в настройки шейдеров между Texture Coordinate и Emission – Separate XYZ, Math, Combine XYZ.

X «Separate XYZ» соединим с Value «Math(Add)».
Value «Math(Add)» c X «Combine XYZ».
Y «Separate XYZ» c Y «Combine XYZ».
Z «Separate XYZ» c Z «Combine XYZ».

B Add выберем тип Multiply и при изменении значения Value, будет меняться цвет картинки.

В значение Value введем frame/10, теперь значение будет изменяться в зависимости от кадра.

В значение Value введем frame/10, теперь значение будет изменяться в зависимости от кадра.

Отрендерим полученный результат.
