Урок 1 – Shape keys

В результате этого урока должна получиться следующая анимация:

Создадим простого монстрика при помощи простых форм.
Вы можете создать похожего персонажа или создать своего.

Перейдем в раздел свойств данных объекта. Во вкладке Shape Keys, нажмем на плюсик.

После этого у нас появится ячейка Basis в которой хранятся данные о положении точек нашего объекта.

Создадим еще один Shape Keys, нажав на плюсик. Затем перейдем в режим редактирования Edit Mode и изменим положение точек. Так, например, сделаем вот такое забавное выражение лица, опустив уголки рта.

Вернемся в Object Mode. Теперь при изменении значения Value и увидим, как меняется выражение нашего персонажа.

Создадим аналогичным способом новые ячейки с другими выражениями.

Также сделаем и с глазами. Создадим несколько вариантов формы глаза.

Так как объект имеет много точек, то для лучшей деформации формы включим данную кнопку.

Создадим несколько вариаций для глаза.

Теперь можем приступить к анимации. Для этого будем выбирать нужную нам ячейку с различными эмоциями, менять значение Value и ставить ключи для создания ключевых кадров.

Например, в 1 кадре выберем FACE_3 со значением Value=0 и установим ключ.

Переместимся в 30 кадр, и для FACE_3 изменим значение Value на 1 и установим ключ.

Таким образом мы можем анимировать любые ячейки и смешивать их между собой.

Таким образом вы можете создать собственную анимацию своего персонажа.

Расставьте ключевые кадры, на свое усмотрение и отрендерите получившуюся сцену.
